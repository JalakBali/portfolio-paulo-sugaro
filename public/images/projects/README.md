# Projects

Contains cover images and thumbnails for individual project case studies displayed in the Projects section.

Recommended naming convention: `<project-slug>-cover.jpg` or `<project-slug>-thumb.jpg`

Examples: `erp-bali-cover.jpg`, `inventory-system-thumb.jpg`

Preferred format: JPG or WebP, minimum 800 px wide, 16:9 aspect ratio recommended.
