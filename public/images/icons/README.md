# Icons

Contains standalone icon files that are not served via an icon library (e.g. custom or brand-specific icons).

Recommended naming convention: `<icon-name>.svg`

Examples: `whatsapp.svg`, `custom-arrow.svg`

Preferred format: SVG. For raster icons, use PNG at 2× resolution (e.g. 64 × 64 px for a 32 px display size).
